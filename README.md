# banking_app
A simple command-line banking application built in Python to practice programming fundamentals such as functions, lists, input validation, and application state management.

## Features

* Create a new bank account
* Secure login using account number and PIN
* Deposit money
* Withdraw money
* Check account balance
* View transaction history
* Logout functionality
* Menu-driven interface

## Technologies Used

* Python 3
* Lists and nested lists
* Functions
* Loops
* Conditional statements
* Exception handling (`try` / `except`)

## Account Structure

Each account stores:

* Account Number
* Account Holder Name
* PIN
* Balance
* Transaction History

## What I Learned

While building this project, I practiced:

* Organizing programs using functions
* Managing program state with a logged-in user
* Working with nested lists
* Validating user input
* Handling exceptions
* Designing a menu-driven application
* Recording transaction history
